DESIGN NOTES FOR FIGMA
═══════════════════════════════════════
- Border radius: 8px for cards, 24px for buttons
- Card shadows: soft, 0 4px 16px rgba(0,0,0,0.08)
- All currency in Philippine Peso (₱)
- Use "Inquire Now" — never "Buy Now" or "Add to Cart"
- Mobile responsive: show mobile frame for homepage
- Spacing system: 8px base grid
- Hover states on all buttons and product cards
- Product cards: white bg, subtle border, 
  blue border on hover
